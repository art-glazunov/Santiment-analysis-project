Приложение определяет тональность отзыва (положительный или отрицательный) и вероятность принадлежности к классу(степень уверенности). 
Можно использовать модель по умолчанию(пайплайн из векторайзера и классификатора) либо любую другую подобную модель, 
предсказывающую классы и вероятности принадлежнрости к классам.
Чтобы работать с моделью необходимо иметь предустановленный интерпретатор python3, а также все модули, импортируемые в коде приложения. 
При построении одной из страниц используется Bootstrap.

Запуск приложения:
1.Открыть коммандную строку (к примеру, через Anaconda3 Powershell Prompt)
2.Перейти в папку приложения cd "Ваш путь"\SA_model_DEMO_AVG
3.Запустить приложение из папки командой python SA_demo_AVG_final.py
4.Открыть ссылку http://127.0.0.1:5000/ в браузере

Использование приложения:
Для получения тональности отзыва нажмите кнопку "Оценить"
Для очистки формы нажмите кнопку "Очистить форму"
Для просмотра параметров используемого классификатора нажмите "Описание модели"
Результаты использования модели можно посмотреть в логе demoAVG_logs.txt

Enjoy!:)
