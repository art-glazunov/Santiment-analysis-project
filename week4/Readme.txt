Данное демо можно использовать для оценки тональности отзывов на английском языке. 
Для запуска нужно:
1. Поместить содержимое папки в вашу рабочую директорию.
2. Запустить ноутбук со скриптом (Python 3).
3. Перейти по ссылке, которая появится внизу после запуска ячейки с кодом.
4. Ввести отзыв на английском и нажать "Оценить".
5. Также имеются скрины экрана, на которых также можно посмотреть работу модели.

P.S. При подгрузке модели и ее работе могут быть проблемы в связи с разными версиями библиотек. 
В таком случае для запуска демо нужно использовать свою модель. 
Enjoy!:)
